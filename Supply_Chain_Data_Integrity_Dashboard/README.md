# Supply Chain Data Integrity & Procurement KPI Dashboard

This project audits procurement data using SQL data quality rules and visualizes
key procurement KPIs using Streamlit.

## KPIs
- On-Time Delivery (OTD)
- Purchase Price Variance (PPV)
- Material Availability

## Stack
SQLite | SQL | Python | Streamlit | Plotly

## Run
pip install -r requirements.txt
streamlit run app.py
